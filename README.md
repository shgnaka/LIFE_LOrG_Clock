# Org Clock Android

Android 端末上で org ファイルの見出しに対して clock 記録を行うアプリです。

## Docs

- 実機インストール手順（WSL2 + USB デバッグ）  
  `docs/install-debug-guide.md`
- 手動スモークテスト手順  
  `docs/manual-smoke-test.md`

## Quick Start

```bash
./gradlew installDebug
```

`installDebug` の詳細は `docs/install-debug-guide.md` を参照してください。
