# Taxonomy Reference

- `ui`: UI flow, presentation logic, notification UX.
- `domain`: use-case orchestration and business rules.
- `data`: repository, persistence, I/O, backup.
- `infra-ci`: CI/CD, workflows, build scripts.
- `tests-quality`: tests, benchmark reliability, quality gates.
- `security-privacy`: auth, permissions, secret handling, sensitive data.
- `docs-ops`: runbooks, incident process, operational docs.
